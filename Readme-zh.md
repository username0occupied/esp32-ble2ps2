# ESP32C3 蓝牙键鼠转 PS2 适配器

基于 ESP32C3 + ESP‑IDF V5.5.0 开发的蓝牙键盘/鼠标转 PS2 协议转换设备，支持双主机、四路 PS2 输出、I2C LCD1602 状态显示、蓝牙配对与电量监控。

## 功能概述
- 蓝牙键盘、蓝牙鼠标 → 双路 PS2 键盘 + 双路 PS2 鼠标
- 同时支持两台 PC 的 PS2 接口
- 四路 PS2 共用一个硬件定时器，资源高效
- LCD1602 实时显示选择状态、初始化、锁键、蓝牙连接、电量、配对码、异常指令

## 硬件引脚定义

### PS2 接口
| 功能        | GPIO | 描述           |
|-------------|------|----------------|
| Key1 CLK    | 19   | 第一台电脑键盘 |
| Key1 DAT    | 13   | 第一台电脑键盘 |
| Mouse1 CLK  | 12   | 第一台电脑鼠标 |
| Mouse1 DAT  | 18   | 第一台电脑鼠标 |
| Key2 CLK    | 2    | 第二台电脑键盘 |
| Key2 DAT    | 3    | 第二台电脑键盘 |
| Mouse2 CLK  | 10   | 第二台电脑鼠标 |
| Mouse2 DAT  | 6    | 第二台电脑鼠标 |

### LCD1602（I2C）
| 功能 | GPIO | 描述        |
|------|------|-------------|
| SCL  | 5    | I2C 时钟线  |
| SDA  | 4    | I2C 数据线  |

## LCD1602 显示规则
第一行：PC1 状态  
第二行：PC2 状态

### 位置定义
| 位置        | 显示字符/值 | 描述说明                                      |
|-------------|-------------|-----------------------------------------------|
| 1           | `>` / 无    | `>` = 当前选中的主机                          |
| 2           | S / s       | S = 键盘就绪；s = 未就绪/异常                 |
| 3           | 1 / ←       | 1 = NumLock 开启；← = NumLock 关闭            |
| 4           | A / a       | A = CapsLock 开启；a = CapsLock 关闭          |
| 5           | - / \|      | - = ScrollLock 开启；\| = ScrollLock 关闭     |
| 6           | K / k       | K = 键盘蓝牙已连接；k = 未连接                |
| 7           | M / m       | M = 鼠标蓝牙已连接；m = 未连接                |
| 8~13        | 配对码      | 蓝牙配对码（优先级低于异常指令）              |
| 12          | K/M + 4位16进制 | 未识别的主机指令（K=键盘，M=鼠标）         |
| 第二行 6~8  | 数字        | 键盘电量百分比（如 100 = 满电）（实现中           |
| 第二行 9~11 | 数字        | 鼠标电量百分比（格式同键盘）（实现中            |
